For driver.c:
to compile
gcc -Wall approaching.c bresenham.c chasing.c classification.c detection.c diver.c following.c queue.c randomwalk.c uMoves.c -lm -o diver

compile in queue_branch
gcc -Wall approaching.c bresenham.c chasing.c classification.c detection.c diver.c following.c randomwalk.c uMoves.c Fifo.c -lm -o diver

to run
following a diagonal path
./diver -V -o following.csv -u "1000 100 diagonal" -v "0 0 following" -t 6000

 chasing a vertical path
./diver -V -o chasing_vertical.csv -u "1000 100 vertical" -v "200 0 chasing" -t 6000

use -d <activity> for detection
with chasing_vertical.csv as an existing file
./diver -V -o chasing_vertical.csv -d chasing
./diver -V -o watch_randomw.csv -d randomwalk
